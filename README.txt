=== Plugin Name ===

Refact WordPress Plugin Boilerplate

== Description ==

    This plugin is a boilerplate for creating a WordPress plugin.

== Installation ==

To utilize this boilerplate, please follow these steps:

1- Run "composer install" command in plugin directory.
2- Search in plugin directory for "plugin-name" and replace it with your plugin name e.g. "`your-plugin-name".
3- Search in plugin directory for "Plugin_Name" and replace it with your plugin name e.g. "Your_Plugin_Name".
4- replace the "plugin-name" with your plugin name in all files names. e,g, "your-plugin-name.php".
